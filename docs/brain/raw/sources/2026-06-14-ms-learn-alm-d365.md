# Best practices for ALM in Dynamics 365 (Microsoft Learn)

> **Imutável.** Referência oficial de ALM para CE.

## Resumo fiel
- Ambientes Dev/Test/UAT/Prod; promover sempre como **managed** (artefato de build).
- Um publisher com prefixo significativo; nunca o `new_` default.
- Source control de solution via unpack (pac/SolutionPackager).
- Environment variables e connection references para portabilidade.
