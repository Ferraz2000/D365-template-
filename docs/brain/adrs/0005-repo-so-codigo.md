# ADR-0005: Repo só código-fonte; solutions fora

- Status: Aceito
- Data: 2026-06-14

## Contexto
Solutions unpacked pesam e poluem o histórico.

## Decisão
Versionar só código-fonte e o padrão; solution exportada como artefato de build (gitignored); promover managed.

## Consequências
Repo enxuto; o padrão de solution fica documentado em docs/architecture, não os arquivos.
